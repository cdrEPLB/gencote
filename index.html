Comment fabriquer une Cote simple :

Les documents d'un lieu documentaire sont rangés à partir de la réflexion menée par le/la documentaliste qui respecte les règles d'une classification pour réaliser les cotes.
Dans un même lieu documentaire. Les documents peuvent être cotés selon une ou plusieurs classifications.
